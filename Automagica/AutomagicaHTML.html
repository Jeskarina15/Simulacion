AUTOMAGICA

Automagica es una plataforma de cdigo abierto Smart Robotic Process Automation (SRPA). Con Automagica, automatizar procesos multiplataforma se convierte en una brisa. Con esta biblioteca de cdigo abierto, queremos proporcionar un contenedor completo y consistente alrededor de las bibliotecas de automatizacin conocidas y menos conocidas.

AUTOMAGICA PPT APPLICATION

En este ejemplo, crearemos una presentacin de PowerPoint totalmente automatizada

In [1]:
from automagica import *
##Iniciando PowerPoint
##Comience abriendo PowerPoint, en este caso le daremos a la instancia el nombre 'presentacin'. 
automagicaPPT = PowerPoint()
##Crear nuestra primera diapositiva
automagicaPPT.add_slide()
##Ahora agregamos algo de contenido. 
automagicaPPT.add_text(text='Hola, esto es power point con automagica', index=1)

In [2]:
##Guardar datos
##Ahora podemos guardar la presentacin, 
##elegimos guardarla como .pdf y como .pptx. Finalmente no olvidemos salir de PowerPoint.

##Primero declaramos el path utilizando la funcin home_path integrada de Automagica para asegurarnos de saber dnde se guarda.
presentation_pptx = home_path('automagicaPPT.pptx')

presentation_pdf = home_path('automagicaPPT.pdf')

# Save pptx
automagicaPPT.save_as(presentation_pptx)
# Save pdf
automagicaPPT.export_to_pdf(presentation_pdf)
##Cerramos el documento
automagicaPPT.quit()
## Verificamos abriendo el contenido en formato PDF
open_file(presentation_pdf)
